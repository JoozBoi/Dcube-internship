# Backend

This folder contains the Express server, API routes, DB/Sequelize setup, and backend-only services.

Structure (intended):
- `src/backend/server.ts`
- `src/backend/routes/*.ts`
- `src/backend/middleware/*.ts`
- `src/backend/db/*.ts`
- `src/backend/lib/*.ts`
- `src/backend/data.js` (seed data)
